########
easy-dsp
########
This is a fork of the original easy-dsp library tailored for the Pyramic
microphone array.


Installation
------------

::

    $ git clone git@github.com:sahandKashani/easy-dsp.git
    $ cd easy-dsp
    $ pip install --user -r requirements.txt .
